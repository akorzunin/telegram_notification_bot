# telegram_notification_bot

## Install

### Docker
 `docker.env` file content:
 ```
 TOKEN=  # telegram bot API token                                                                    
 BINANCE_API_KEY=                                         
 BINANCE_SECRET_KEY=                                                                                           
 ENDPOINT=  # API documentation page                                                                                                        
 PORT=  # API documentation page
 ```
 > sudo docker-compose -f "docker-compose.yml" up -d --build
